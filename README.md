# \# 🧠 Dementia Care AI

# 

# AI 기반 음성·텍스트 분석을 통해 사용자의 인지 기능 변화를 모니터링하고

# 치매 위험도를 조기 예측하는 안드로이드 애플리케이션입니다.

# 

# ---

# 

# \## 📱 App Info

# 

# \* \*\*Platform\*\*: Android

# \* \*\*Language\*\*: Kotlin

# \* \*\*Version\*\*: 0.1.0 (Prototype)

# \* \*\*Min SDK\*\*: 24

# \* \*\*Target SDK\*\*: 36

# 

# ---

# 

# \## 🚀 주요 기능

# 

# \### 🎤 1. 음성 기반 분석

# 

# \* 사용자 음성 녹음 및 저장

# \* 발화 속도, 침묵 구간, 반복 발화 분석

# \* pitch, jitter, shimmer 기반 음성 특징 추출

# 

# \### 📝 2. 텍스트 분석

# 

# \* STT(Speech-to-Text)를 통한 텍스트 변환

# \* 어휘 다양성, 문장 길이, 반복 표현 분석

# \* 의미 연결성 및 언어 패턴 변화 감지

# 

# \### 🧠 3. 기억 일치도 평가

# 

# \* 과거 대화 데이터와 현재 응답 비교

# \* 회상 정확도 기반 인지 기능 평가

# 

# \### 📊 4. 위험도 분석 및 시각화

# 

# \* 음성 + 텍스트 + 기억 데이터 통합 분석

# \* 일별/주별 인지 기능 변화 그래프 제공

# 

# \### 🔔 5. 알림 및 모니터링

# 

# \* 정기 자가 점검 알림

# \* 이상 징후 발생 시 경고 알림

# \* 보호자 연동 기능 (예정)

#
